# My Dream Destination

This is a simple static website about a dream travel destination: **Japan**.
link to my website :https://dhruv1176.github.io/My-Dream-Destination/
## Files

- `index.html` - Main page describing Japan, including:
  - about section
  - top attractions
  - things to do
  - travel itinerary table
  - useful links
- `public/contact.html` - Contact page with a brief message for visitors.
- `asset/images/mount fuji.jpg` - Featured image used on the main page.

## Features

- Uses plain HTML and inline table styling
- Includes internal navigation to the `contact.html` page
- Uses an external link to the Wikipedia page for Japan

## How to view

1. Open `index.html` in your browser.
2. Click the "contact us" link to view the contact page.

## Notes

- The content is focused on Japan's culture, attractions, and travel highlights.
- You can expand the site by adding CSS styling in `asset/css/` or by creating more pages.
